## AI Portfolio Assistant

The standout feature вЂ” a chatbot that knows everything about my CV, projects, and experience.

- Powered by **DeepSeek LLM** with bilingual support (Vietnamese & English)
- Three response styles: Brief, Detailed, and Auto-fit
- Upload a Job Description and the AI analyzes profile-match
- Type `/` to access quick commands
- Export conversations to PDF

---

## Highlights

- Brutalist Design вЂ” dark theme with neon green (#CCFF00) accents
- Interactive 3D elements (Three.js + React Three Fiber)
- Smooth animations with Framer Motion and GSAP
- Rich text contact form with Slate.js and email preview
- Fully responsive across all devices
- Optional sound experience

**Sections:** Hero, About, Work History, Portfolio (9+ projects), Certifications, Videos, Contact

## Getting Started

```bash
git clone https://github.com/xuanhai0913/My-Portfolio-NXH.git
cd My-Portfolio-NXH
npm install
cp .env.example .env
npm start
# Open http://localhost:3000
```
